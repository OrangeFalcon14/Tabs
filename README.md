## Installation
### Linux
#### Download Executable
- Go to the [Releases](https://github.com/OrangeFalcon14/Tabs/releases/) page and download the latest release
- Run the executable file
#### Build from source:
- Install Python
- Install PyGTK if it isn't already
- Clone this repo
- Open your terminal and go to the directory where the clone is stored
- Run `pip install -r requirements.txt`
- Run `python3 main.py`
